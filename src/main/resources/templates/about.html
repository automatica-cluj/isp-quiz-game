How This Quiz Works

Unlimited Attempts

You can take quizzes as many times as you want. Each run is independent, so feel free to practice until you feel confident.

Usernames & Scores

There is no authentication layer on top of usernames. You may use any name; if you achieve a higher score with the same username, it overwrites the previous entry. Otherwise, the older (better) score stays visible. The leaderboard currently shows only the latest best score per user.

Scoring Behavior

The scoring algorithm may change over time as the course evolves. Pay attention to updates in announcements or release notes.

Randomized Questions & Answers

Each quiz run loads questions from the shared text bank and randomizes both the question selection and the order of answers. The total bank size and the subset delivered per session depend on the current configuration.

Timing & Bonus Time

The quiz is time-limited and the remaining time is shown on the quiz page. When the optional bonus-time feature is enabled you gain an extra X seconds every time you answer correctly.

Report Bugs or Suggest Features

Please file issues (except question content corrections) in the public tracker: https://github.com/automatica-cluj/isp-quiz-game/issues .
